Как делать кнопки

Создание базовых кнопок

Для создания самой кнопки указываем элемент <а> или <button> с классом btn

Пример:

<a class="btn">button</a>

button

Это самая простая кнопка, она среднего размера и не имеет заднего фона и "ободка" (p.s. не знаю как outline обозвать)

Теперь добавим фон дополнительно вписав в класс btn-primary

Пример:

<a class="btn btn-primary">primary button</a>

primary button

Или же можно сделать кнопку с "оботком"

<a class="btn btn-outline-primary">outline primary button</a>

outline primary button

Изменения размера кнопок

В данной итерации существуют только три типовых размера кнопок - большая, средняя и маленькая
По дефолту при создании кнопки без явного указания размера ей задается стандартное среднее значение

Для создания кнопки другого размера дополнительно вписываем в класс btn-lg (расшиф. button long) или btn-sm (расшиф. button small)

Пример:

<a class="btn btn-primary btn-lg">long primary button</a>

long primary button

<a class="btn btn-primary btn-sm">small primary button</a>

small primary button

Эти "теги" размера применимы ко всем видам кнопок, т.е. - простым, с фоном и с ободком


Добавление иконки

Во все вышеперечисленные кнопки можно вставить иконку слева или справа относительно текста.

Для этого в тело строчного элемента <a> после текста кнопки вписываем данную строку

<svg class="icon-file-name 64px"></svg>

Где "icon-file-name" - название файла иконки (без указания формата .svg!) в папке src/img/icons
А 64px - размер с которым иконка будет отображена

Пример:

<a class="btn btn-sm btn-primary">
primary button
<svg class="fluent_checkmark-starburst-16-regular 64px"></svg>
</a>

primary button

Для размещения иконки слева относительно текста введите в класс btn-icon-left-margin

<a class="btn btn-sm btn-primary btn-icon-left-margin">
primary button
<svg class="fluent_checkmark-starburst-16-regular 64px"></svg>
</a>

primary button
